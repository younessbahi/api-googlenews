Get started

API Endpoint:
    https://api-googlenews.herokuapp.com/

An alternative to the official google api that allows you to retrieve 2 to 4 times more news headlines compared to similar existing apis. When you search by topic, similar news headlines share the same [item_guid]

You don't need an API KEY to use this API. Please contact me at DAyounessbahi@gmail.com if you have any questions or suggestions.

get headlines from custom search


# Here is a curl example
curl -X GET "https://api-googlenews.herokuapp.com/search?q=russia&country=US" -H "accept: */*"

To get news headlines from custom search you need to make a POST call to the following url :
https://api-googlenews.herokuapp.com/search?

get list of available topics


# Here is a curl example
curl -X GET "https://api-googlenews.herokuapp.com/topic/list" -H "accept: */*"

To get the list of available topics you need to make a POST call to the following url :
https://api-googlenews.herokuapp.com/topic/list

Errors

This API uses the following error codes:

Error Code Meaning
1001 This error appears when your input is too short to be a country code or country name.
1002 This error appears if you pass an invalid country name.
1003 This error appears if you pass an invalid country code.
1004 Unknown or invalid category. This error appears when you pass a category other than the predefined ones.
1005 This error appears when there is no result found for your search.
